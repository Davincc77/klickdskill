# `docs/community/` — community & dev-rel pack

Lightweight entry points for developers who land on the repo from clone traffic, search, or referral.

| File | Purpose |
|---|---|
| [`TRY_IT.md`](./TRY_IT.md) | 5-minute path from clone to "I loaded a `.klickd` file in code". |
| [`FEEDBACK.md`](./FEEDBACK.md) | The five questions we would like answered. Where to put answers. |
| [`DISCUSSION_PROMPTS.md`](./DISCUSSION_PROMPTS.md) | **Drafts** for the first GitHub Discussions seed posts. **Each requires Vince approval before posting.** |

This directory is **documentation only**. It triggers no release, ships no schema change, ships no `.klickd` artefact, and does not enable GitHub Discussions on its own.
